# Определение оптимальной модели эмбеддингов и способа чанкинга для русскоязычной RAG-системы в браузере

**Цель работы:** определить модель эмбеддингов и способ нарезки текста на чанки, обеспечивающие наилучшее качество поиска релевантной информации по документам на русском языке при локальном выполнении в браузере (в контексте RAG-системы MeMemo).
